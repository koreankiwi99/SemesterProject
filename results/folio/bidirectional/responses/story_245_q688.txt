Premises: In superhero movies, the good guys always win.
The Surprising Adventures of Sir Digby Chicken Caesar is a superhero movie.
Good guys fight bad guys and vice versa.
Sir Digby fights his nemesis.
If a superhero movie is named after a character, that character is a good guy.
The Surprising Adventures of Sir Digby Chicken Caesar is named after Sir Digby.
If somebody wins a fight, the person they are fighting does not win.
If a superhero movie is named after a character, that character is in the movie.
Conclusion: Sir Digby’s nemesis does not win.

axiom Person : Type
axiom Movie : Type

axiom SirDigby : Person
axiom Nemesis : Person
axiom SASDCC : Movie

axiom SuperheroMovie : Movie → Prop
axiom InMovie : Person → Movie → Prop
axiom GoodGuy : Person → Prop
axiom BadGuy : Person → Prop
axiom Fights : Person → Person → Prop
axiom NamedAfter : Movie → Person → Prop
axiom Wins : Person → Prop

-- Premises
-- In superhero movies, the good guys always win.
axiom good_guys_always_win_in_superhero_movies :
  ∀ (m : Movie) (c : Person), SuperheroMovie m → InMovie c m → GoodGuy c → Wins c

-- The Surprising Adventures of Sir Digby Chicken Caesar is a superhero movie.
axiom SASDCC_is_superhero : SuperheroMovie SASDCC

-- Good guys fight bad guys and vice versa.
axiom good_fight_bad : ∀ (a b : Person), GoodGuy a → BadGuy b → Fights a b
axiom bad_fight_good : ∀ (a b : Person), BadGuy a → GoodGuy b → Fights a b

-- Sir Digby fights his nemesis.
axiom fights_nemesis : Fights SirDigby Nemesis

-- If a superhero movie is named after a character, that character is a good guy.
axiom named_after_implies_good_guy :
  ∀ (m : Movie) (p : Person), SuperheroMovie m → NamedAfter m p → GoodGuy p

-- The Surprising Adventures of Sir Digby Chicken Caesar is named after Sir Digby.
axiom SASDCC_named_after_SirDigby : NamedAfter SASDCC SirDigby

-- If somebody wins a fight, the person they are fighting does not win.
axiom win_fight_implies_opponent_not_win :
  ∀ (a b : Person), Fights a b → Wins a → ¬ Wins b

-- If a superhero movie is named after a character, that character is in the movie.
axiom named_after_implies_in_movie :
  ∀ (m : Movie) (p : Person), SuperheroMovie m → NamedAfter m p → InMovie p m

-- Conclusion: Sir Digby’s nemesis does not win.
theorem nemesis_does_not_win : ¬ Wins Nemesis := by
  have hIn : InMovie SirDigby SASDCC :=
    named_after_implies_in_movie SASDCC SirDigby SASDCC_is_superhero SASDCC_named_after_SirDigby
  have hGood : GoodGuy SirDigby :=
    named_after_implies_good_guy SASDCC SirDigby SASDCC_is_superhero SASDCC_named_after_SirDigby
  have hWinSir : Wins SirDigby :=
    good_guys_always_win_in_superhero_movies SASDCC SirDigby SASDCC_is_superhero hIn hGood
  have hFight : Fights SirDigby Nemesis := fights_nemesis
  exact win_fight_implies_opponent_not_win SirDigby Nemesis hFight hWinSir

Verification Success: True
