Premises: None of the students who currently work in the library are from the computer science department. 
All of the students with part-time jobs offered by the university currently work in the library. 
All of the students who are taking the database course are from the computer science department. 
All of the students taking a class with Professor David are taking the database course. 
All students working in the lab are taking a class with Professor David.
James is a student working in the lab. 
James doesn't work in the lab or have a part-time job offered by the university.
Conclusion: James has a part-time job offered by the university.

PROVE TRUE ATTEMPT
======================================================================

axiom Person : Type
axiom James : Person

axiom Student : Person → Prop
axiom WorksInLibrary : Person → Prop
axiom FromCS : Person → Prop
axiom HasUnivPT : Person → Prop
axiom TakingDatabase : Person → Prop
axiom TakingWithDavid : Person → Prop
axiom WorksInLab : Person → Prop

-- Premises:
-- 1. None of the students who currently work in the library are from the CS department.
axiom p1 : ∀ x : Person, Student x ∧ WorksInLibrary x → ¬ FromCS x
-- 2. All of the students with part-time jobs offered by the university currently work in the library.
axiom p2 : ∀ x : Person, Student x ∧ HasUnivPT x → WorksInLibrary x
-- 3. All of the students who are taking the database course are from the computer science department.
axiom p3 : ∀ x : Person, Student x ∧ TakingDatabase x → FromCS x
-- 4. All of the students taking a class with Professor David are taking the database course.
axiom p4 : ∀ x : Person, Student x ∧ TakingWithDavid x → TakingDatabase x
-- 5. All students working in the lab are taking a class with Professor David.
axiom p5 : ∀ x : Person, Student x ∧ WorksInLab x → TakingWithDavid x
-- 6. James is a student working in the lab.
axiom p6 : Student James ∧ WorksInLab James
-- 7. James doesn't work in the lab or have a part-time job offered by the university.
-- Interpreted as: James neither works in the lab nor has a university part-time job.
axiom p7 : ¬ WorksInLab James ∧ ¬ HasUnivPT James

-- Conclusion: James has a part-time job offered by the university.
theorem conclusion : HasUnivPT James := by
  have hW : WorksInLab James := And.right p6
  have hNW : ¬ WorksInLab James := And.left p7
  have hFalse : False := hNW hW
  exact False.elim hFalse
