<lean>
axiom obj : Type
axiom StrangerThings : obj
axiom BlackMirror : obj
axiom IsNetflixShow : obj → Prop
axiom Popular : obj → Prop
axiom BingeWatches : obj → Prop
axiom Downloads : obj → Prop
axiom SharesWithLisa : obj → Prop

axiom P1 : IsNetflixShow StrangerThings
axiom P2 : Popular StrangerThings
axiom P3 : ∀ (x : obj), IsNetflixShow x → Popular x → BingeWatches x
axiom P4 : ∀ (x : obj), IsNetflixShow x → (BingeWatches x ↔ Downloads x)
axiom P5 : ¬ Downloads BlackMirror
axiom P6 : IsNetflixShow BlackMirror
axiom P7 : ∀ (x : obj), IsNetflixShow x → BingeWatches x → SharesWithLisa x

theorem share_ST : SharesWithLisa StrangerThings := by
  have binge_ST : BingeWatches StrangerThings := P3 StrangerThings P1 P2
  exact P7 StrangerThings P1 binge_ST
</lean>

ANSWER: True

Explanation:  
The conclusion follows directly from:  
1. `StrangerThings` is a popular Netflix show (P1, P2)  
2. Popular Netflix shows are binge-watched (P3 → `BingeWatches StrangerThings`)  
3. Binge-watched Netflix shows are shared with Lisa (P7 → `SharesWithLisa StrangerThings`)  

The Black Mirror information is unused but consistent (via P4/P5/P6 proving `¬BingeWatches BlackMirror`), demonstrating no contradiction.
