The goal is to disprove the statement: "If something is a household appliance, it sucks." From the premises, we know that a duster is explicitly defined as a household appliance that does not suck. This directly contradicts the universal claim that all household appliances suck. Therefore, the statement is false.

<lean>
axiom obj : Type
axiom Sucks : obj → Prop
axiom HouseholdAppliance : obj → Prop

-- Premises: Categories
axiom Plunger : obj → Prop
axiom Vampire : obj → Prop
axiom Vacuum : obj → Prop

-- Premises: Properties
axiom plungers_suck : ∀ x, Plunger x → Sucks x
axiom vampires_suck : ∀ x, Vampire x → Sucks x
axiom vacuums_suck : ∀ x, Vacuum x → Sucks x

-- Premises: Specific objects
axiom Space : obj
axiom space_vacuum : Vacuum Space

axiom Duster : obj
axiom duster_appliance : HouseholdAppliance Duster
axiom duster_no_suck : ¬Sucks Duster

-- Disproof of the universal claim
theorem appliance_not_always_sucks : 
    ¬(∀ x, HouseholdAppliance x → Sucks x) := 
  by
    intro H  -- Assume the universal claim
    have duster_sucks := H Duster  -- Specialize to duster
    exact duster_no_suck (duster_sucks duster_appliance)  -- Contradiction
</lean>

ANSWER: False
