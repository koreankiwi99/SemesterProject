- Every printed text is analog: `∀ x, PrintedText x → AnalogMedia x`
- All streaming services are digital: `∀ x, StreamingService x → DigitalMedia x`
- Hardcover books are printed texts: `∀ x, HardcoverBook x → PrintedText x`
- If 1984 is a streaming service, then it is a hardcover book: `StreamingService NineteenEightyFour → HardcoverBook NineteenEightyFour`

To prove "1984 is printed text", we must show `PrintedText NineteenEightyFour`. However:
- The only path to "printed text" requires 1984 to be a hardcover book (via `hardcover_printed`).
- The hardcover book property is only triggered if 1984 is a streaming service (via `conditional`).

  2. Hence cannot be analog (`no_digital_analog`)
  3. But also is printed text (via hardcover → printed → analog), requiring analog media.
