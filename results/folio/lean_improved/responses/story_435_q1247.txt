Premises: None of the students who currently work in the library are from the computer science department. 
All of the students with part-time jobs offered by the university currently work in the library. 
All of the students who are taking the database course are from the computer science department. 
All of the students taking a class with Professor David are taking the database course. 
All students working in the lab are taking a class with Professor David.
James is a student working in the lab. 
James doesn't work in the lab or have a part-time job offered by the university.
Conclusion: James takes the database course.

==================================================
<lean>
universe u

constant Person : Type u

constant Student : Person → Prop
constant WorksInLibrary : Person → Prop
constant FromCSDept : Person → Prop
constant PartTimeUniv : Person → Prop
constant TakesDatabase : Person → Prop
constant TakesClassWithDavid : Person → Prop
constant WorksInLab : Person → Prop

constant james : Person

-- Premises
axiom h1 : ∀ x : Person, Student x → WorksInLibrary x → ¬ FromCSDept x
axiom h2 : ∀ x : Person, Student x → PartTimeUniv x → WorksInLibrary x
axiom h3 : ∀ x : Person, Student x → TakesDatabase x → FromCSDept x
axiom h4 : ∀ x : Person, Student x → TakesClassWithDavid x → TakesDatabase x
axiom h5 : ∀ x : Person, Student x → WorksInLab x → TakesClassWithDavid x
axiom h6a : Student james
axiom h6b : WorksInLab james
axiom h7 : ¬ (WorksInLab james ∨ PartTimeUniv james)

-- Conclusion
theorem james_takes_database : TakesDatabase james := by
  have hcd : TakesClassWithDavid james := h5 james h6a h6b
  exact h4 james h6a hcd
</lean>
