Premises: Everyone who can register to vote in the United States can participate in the 2024 United States presidential election.
If someone has United States citizenship, then they can register to vote in the United States.
A person either has United States citizenship or Taiwanese citizenship.
No Russian Federation officials hold Taiwanese citizenship. 
Vladimir neither holds Taiwanese citizenship nor is he a manager at Gazprom. 
Ekaterina she can register to vote in the United States, or she is a Russian federation official.
Conclusion: Ekaterina can participate in the 2024 United States presidential election or is a manager at Gazprom.

-- Entities and predicates
constant Person : Type

constant RegisterUS : Person → Prop
constant Participate2024 : Person → Prop
constant USCitizen : Person → Prop
constant TaiwaneseCitizen : Person → Prop
constant RussianOfficial : Person → Prop
constant ManagerGazprom : Person → Prop

constant Ekaterina : Person
constant Vladimir : Person

-- Premises
axiom ax1 : ∀ x : Person, RegisterUS x → Participate2024 x
axiom ax2 : ∀ x : Person, USCitizen x → RegisterUS x
axiom ax3 : ∀ x : Person, USCitizen x ∨ TaiwaneseCitizen x
axiom ax4 : ∀ x : Person, RussianOfficial x → ¬ TaiwaneseCitizen x
axiom ax5 : ¬ TaiwaneseCitizen Vladimir ∧ ¬ ManagerGazprom Vladimir
axiom ax6 : RegisterUS Ekaterina ∨ RussianOfficial Ekaterina

-- Conclusion: Ekaterina can participate in the 2024 US presidential election or is a manager at Gazprom.
theorem goal : Participate2024 Ekaterina ∨ ManagerGazprom Ekaterina :=
by
  -- From ax6, do case analysis
  cases ax6 with
  | inl hReg =>
      -- If she can register, then she can participate (ax1)
      exact Or.inl (ax1 Ekaterina hReg)
  | inr hRO =>
      -- If she is a Russian official, then not Taiwanese (ax4)
      have hNotTW : ¬ TaiwaneseCitizen Ekaterina := ax4 Ekaterina hRO
      -- From ax3, she is either US or Taiwanese citizen
      have hDisj : USCitizen Ekaterina ∨ TaiwaneseCitizen Ekaterina := ax3 Ekaterina
      -- Resolve the disjunction using ¬Taiwanese to get USCitizen
      have hUS : USCitizen Ekaterina :=
        Or.elim hDisj
          (fun hUS => hUS)
          (fun hTW => False.elim (hNotTW hTW))
      -- Then she can register (ax2), hence can participate (ax1)
      have hReg2 : RegisterUS Ekaterina := ax2 Ekaterina hUS
      have hPart : Participate2024 Ekaterina := ax1 Ekaterina hReg2
      exact Or.inl hPart
