Context:
In a zoo, lions and tigers are carnivores. Normally, carnivores in the zoo are fed raw meat. If an animal has a mane and roars loudly, visitors find it majestic. If a big cat has stripes and hunts at night, visitors are intrigued. Visitors find either an animal with a mane majestic or a big cat that hunts at night intriguing, or both. If the lion is majestic, then the tiger is intriguing. If the tiger is fed raw meat, then the lion is majestic.

Question:
Can we conclude if the lion is not majestic then the tiger is fed raw meat?

axiom stripes_and_night_implies_intriguing : ∀ x : Animal, BigCat x ∧ HasStripes x ∧ HuntsAtNight x → Intriguing x
axiom either_mane_majestic_or_bigcat_night_intriguing :
  (∀ x : Animal, HasMane x → Majestic x) ∨ (∀ x : Animal, BigCat x ∧ HuntsAtNight x → Intriguing x)
axiom lion_majestic_implies_tiger_intriguing : Majestic lion → Intriguing tiger
axiom tiger_fed_implies_lion_majestic : FedRawMeat tiger → Majestic lion

-- Goal: If the lion is not majestic then the tiger is fed raw meat.
theorem conclusion : ¬ Majestic lion → FedRawMeat tiger := by
  have hfed : FedRawMeat tiger := carnivores_are_fed tiger tiger_is_carnivore
  intro _
  exact hfed
