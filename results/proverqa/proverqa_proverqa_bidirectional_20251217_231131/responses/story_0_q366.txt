Premises: Keanu has sharp hooves. All active deer have strong senses and good vision. All deer with powerful muscles and a lightweight build can run quickly. Keanu is calm. Florence is active. If a deer has keen senses, it may not necessarily run quickly, and vice versa. A deer either has a gentle nature or explores the wild, but not both is not necessarily true. All deer that live in the forest have keen senses. Keanu is not playful. Any deer that is surefooted or can leap far can explore the wild. Zayd has sharp hooves. Any deer that has sharp hooves and agile legs can run quickly. If Keanu has large ears or good vision, then he can live in the forest. Keanu has strong senses. Zayd is not playful. Keanu is active. If Keanu is calm, then he is either playful or has a gentle nature, but not both. Zayd has strong senses. If Keanu is fleet, then he is either capable of leaping far or is powerful, but not both. If Keanu has large antlers, then he is either capable of leaping far or has agile legs, but not both.
Conclusion: Keanu does not have large antlers.

-- Premises
axiom sh_Keanu : SharpHooves Keanu
axiom all_active_have_strong_and_good : ∀ x : Deer, Active x → StrongSenses x ∧ GoodVision x
axiom muscles_lightweight_run : ∀ x : Deer, PowerfulMuscles x ∧ LightweightBuild x → RunQuickly x
axiom calm_Keanu : Calm Keanu
axiom active_Florence : Active Florence
-- We formalize the "may not necessarily ... and vice versa" as mutual exclusivity
axiom keen_implies_not_run : ∀ x : Deer, KeenSenses x → ¬ RunQuickly x
axiom run_implies_not_keen : ∀ x : Deer, RunQuickly x → ¬ KeenSenses x
-- We use "but not both" (mutual exclusion) between gentle nature and exploring the wild
axiom not_both_gentle_explores : ∀ x : Deer, ¬ (GentleNature x ∧ ExploresWild x)
axiom forest_implies_keen : ∀ x : Deer, LiveInForest x → KeenSenses x
axiom not_playful_Keanu : ¬ Playful Keanu
axiom surefooted_or_leap_implies_explores : ∀ x : Deer, (Surefooted x ∨ LeapFar x) → ExploresWild x
axiom sh_Zayd : SharpHooves Zayd
axiom sharp_and_agile_implies_run : ∀ x : Deer, SharpHooves x ∧ AgileLegs x → RunQuickly x
axiom ears_or_good_implies_forest_Keanu : (LargeEars Keanu ∨ GoodVision Keanu) → LiveInForest Keanu
axiom strong_Keanu : StrongSenses Keanu
axiom not_playful_Zayd : ¬ Playful Zayd
axiom active_Keanu : Active Keanu
axiom calm_implies_xor_playful_gentle_Keanu : Calm Keanu → ((Playful Keanu ∨ GentleNature Keanu) ∧ ¬ (Playful Keanu ∧ GentleNature Keanu))
axiom strong_Zayd : StrongSenses Zayd
axiom fleet_implies_xor_leap_or_powerful_Keanu : Fleet Keanu → ((LeapFar Keanu ∨ Powerful Keanu) ∧ ¬ (LeapFar Keanu ∧ Powerful Keanu))
axiom antlers_implies_xor_leap_or_agile_Keanu : LargeAntlers Keanu → ((LeapFar Keanu ∨ AgileLegs Keanu) ∧ ¬ (LeapFar Keanu ∧ AgileLegs Keanu))

theorem keanu_no_large_antlers : ¬ LargeAntlers Keanu := by
  intro hAntlers
  -- From activity, Keanu has good vision
  have hStrongAndGood := all_active_have_strong_and_good Keanu active_Keanu
  have hGood : GoodVision Keanu := And.right hStrongAndGood
  -- Therefore he lives in the forest, hence has keen senses
  have hForest : LiveInForest Keanu := ears_or_good_implies_forest_Keanu (Or.inr hGood)
  have hKeen : KeenSenses Keanu := forest_implies_keen Keanu hForest
  -- From calm and not playful, Keanu is gentle
  have hX := calm_implies_xor_playful_gentle_Keanu calm_Keanu
  have hGentle : GentleNature Keanu :=
    Or.elim (And.left hX)
      (fun hPlay => False.elim (not_playful_Keanu hPlay))
      (fun hGent => hGent)
  -- From large antlers, either leap far or have agile legs
  have hDisj := And.left (antlers_implies_xor_leap_or_agile_Keanu hAntlers)
  cases hDisj with
  | inl hLeap =>
      have hExplores : ExploresWild Keanu := surefooted_or_leap_implies_explores Keanu (Or.inr hLeap)
      exact (not_both_gentle_explores Keanu) ⟨hGentle, hExplores⟩
  | inr hAgile =>
      have hRun : RunQuickly Keanu := sharp_and_agile_implies_run Keanu ⟨sh_Keanu, hAgile⟩
      have hNotKeen : ¬ KeenSenses Keanu := run_implies_not_keen Keanu hRun
      exact hNotKeen hKeen
