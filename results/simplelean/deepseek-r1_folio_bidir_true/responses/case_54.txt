The statement "Robert Lewandowski is a soccer player" follows directly from the premises. Specifically:
- Premise 1: Robert Lewandowski is a striker
- Premise 2: Strikers are soccer players (universal quantification)
Applying Premise 2 to Premise 1 yields the conclusion. The premises about leaving Bayern Munchen and the implication about no longer playing for the team are irrelevant to this conclusion.

<lean>
axiom obj : Type
axiom Lewandowski : obj
axiom BayernMunchen : obj
axiom Striker : obj → Prop
axiom SoccerPlayer : obj → Prop
axiom Leaves : obj → obj → Prop
axiom PlaysFor : obj → obj → Prop
axiom P1 : Striker Lewandowski
axiom P2 : ∀ x : obj, Striker x → SoccerPlayer x
axiom P3 : Leaves Lewandowski BayernMunchen
axiom P4 : ∀ (x y : obj), Leaves x y → ¬ PlaysFor x y

theorem lewandowski_soccer_player : SoccerPlayer Lewandowski :=
  P2 Lewandowski P1
</lean>

ANSWER: True
