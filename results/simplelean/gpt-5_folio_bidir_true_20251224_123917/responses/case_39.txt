axiom DesignByMax : obj → Prop
axiom Adores : obj → Prop
axiom Interesting : obj → Prop
axiom Brutalist : obj → Prop
axiom Evocative : obj → Prop
axiom Dreamy : obj → Prop
axiom ZahaStyle : obj → Prop
axiom KellyStyle : obj → Prop
axiom Style : obj → Prop

-- Premises
-- 1. All of Zaha Hadid's design styles that Max adores have interesting geometries.
axiom P1 : ∀ x : obj, ZahaStyle x ∧ Adores x → Interesting x

-- 2. No brutalist buildings that Max adores have interesting geometries.
axiom P2 : ∀ x : obj, Brutalist x ∧ Adores x → ¬ Interesting x

-- 3. Every style that Max adores is either Zaha Hadid's design style or Kelly Wearstler's design style.
axiom P3 : ∀ x : obj, Style x ∧ Adores x → (ZahaStyle x ∨ KellyStyle x)

-- 4. All of Kelly Wearstler's design styles that Max adores are evocative.
axiom P4 : ∀ x : obj, KellyStyle x ∧ Adores x → Evocative x

-- 5. All of Kelly Wearstler's design styles that Max adores are dreamy.
axiom P5 : ∀ x : obj, KellyStyle x ∧ Adores x → Dreamy x

-- 6. If a design by Max that he adores has interesting geometries, then it is a brutalist building and evocative.
axiom P6 : ∀ x : obj, DesignByMax x ∧ Adores x ∧ Interesting x → Brutalist x ∧ Evocative x

-- Goal to evaluate: ∃ x, DesignByMax x ∧ Brutalist x
-- There is no premise guaranteeing the existence of any design by Max, or any adored item,
-- so we cannot derive existence of a brutalist design by Max from the given premises.
-- Hence the statement cannot be proven from the premises.
